## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/hgbaodev/chatapp.git
   ```

2. Change into the project directory:

   ```bash
   cd chatapp
   ```

3. Run app
   ```bash
   npm install
   ```

4. Run app
   ```bash
   npm run dev
   ```
